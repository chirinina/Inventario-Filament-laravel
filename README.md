## Instalacion del proyecto

1. Instalacion de dependencias
```bash
$   composer install
```

2. Crear base de datos
```bash
$   docker compose up -d
```

3. Renombrar ``.env.example`` a ``.env``

4. Crear usuario de filament
```bash
$   php artisan make:filament-user
```
